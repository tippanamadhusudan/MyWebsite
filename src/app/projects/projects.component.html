PROJECT : Human-Connection

DESCRIPTION :

  • Added a user role change feature to admin.
  • Admin cannot change their role, but can change the roles of other users.
  • On the roles column, admin can click on the role of the user to get a dropdown window and select the role to assign to user.
  • All the roles in the dropdown window and users role is aquired dinamically.
  • After changing the role, the data will be sent to backend where it will be processed

  • Vue.js

  • CSS

  • Apollo
